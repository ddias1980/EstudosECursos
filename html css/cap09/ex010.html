Tipos de ordenações possíveis

  • 1 = numeros
  • A = Letras maiusculas
  • a = letras minusculas
  • i = romanos minusculas
  • I romanos maiusculas

Listas de Definições

HTML
Linguagem de marcação para a criação e conteúdo de um site
CSS
Linguagem de marcação para design de um site
JavaScript
Linguagem de programação para a criação de interatividade com um site.

Listas de definições foram usados em cima

É preciso usar as TAG DL (definition list) para abrir as listas e alternar as listas entre DT (assunto da esquerda) com DD (definição à direita

Ex:

        
            <dl>
                <dt>HTML</dt>
                <dd>Linguagem de marcação para a criação e conteúdo de um site</dd>
                <dt>CSS</dt>
                <dd>Linguagem de marcação para design de um site</dd>
                <dt>JavasCript</dt>
                <dd>Linguagem de programação para a criação de interatividade com um site.</dd>
            </dl>
