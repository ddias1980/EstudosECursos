13 - Funções auto executáveis

Variáveis locais e globais

Toda variável criada sem o var se torna global e será usada fora da função. Isso não é legal. Toda função deve ter variáveis locais, criadas sempre com o var. 

        
            function variaveis(){
                var x = 10;
                var y = 20;
                console.log (x + y)
            }
            variaveis();
            console.log(y);
        
        
 A variável retornará um erro nesse contexto

Funções autoexecutáveis

Para uma função auto executável, basta adicionar a função entre parenteses, dessa forma não é preciso mais chama-la

        
            (function variaveis(){
                var x = 10;
                var y = 20;
                console.log (x + y)
            })()

Parâmetros e o parametro arguments

Voce já sabe passar paraâmetros, então apenas decore o parâmtro arguments. Arguments recebe os parametros e transforma um array. Voce pode acessar cada valor passado como argumentos percorrendo o array

        
            function variaveis(qualquernomenaoimporta){
                var cont = 0
                var x = 0
                while (cont < arguments.length){
                    x += arguments[cont];
                    cont++;
                }
                console.log(x)
            }
            variaveis(10,30, 20,2,10,5)
        
        forma alternativa
