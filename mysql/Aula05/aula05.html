Aula 05 - Inserindo dados na tabela

Comando DDL

DDL é DATA Definition Language. Comandos para definir bancos de dados, tabelas e afins. O comando CREATE TABLE, CREATE DATABASE.

DML - Data Manipulation Language - O comando INSERT que usaremos hoje é DML, ou seja, manipulamos os dados com ele.


        INSERT INTO pessoas
        (id, nome, nascimento, sexo, peso, altura, nacionalidade)
        VALUES
        ('1', 'Godofredo', '1980-07-10', 'M', '78.5', '1.80', 'Brasil');

Também podemos criar a tabela sem especificar o ID conforme abaixo


        INSERT INTO pessoas
        (nome, nascimento, sexo, peso, altura, nacionalidade)
        VALUES
        ('Monica Santos', '1982-11-19', 'F', '76.5', '1.75', 'Brasil');

Veja os dados da tabelas com o comando SELECT


        SELECT * FROM pessoas
        +----+---------------+------------+------+-------+--------+---------------+
        | id | nome          | nascimento | sexo | peso  | altura | nacionalidade |
        +----+---------------+------------+------+-------+--------+---------------+
        |  1 | Godofredo     | 1980-07-10 | M    | 78.50 |   1.80 | Brasil        |
        |  2 | Monica Santos | 1982-11-19 | F    | 76.50 |   1.75 | Brasil        |
        +----+---------------+------------+------+-------+--------+---------------+

Não precisa dizer as colunas se voce se seguir as ordens e para os campos auto preenchidos, apenas preencha com default


        INSERT INTO pessoas
        VALUES
        (DEFAULT, 'David Dias', '1980-07-10', 'M', '99.5', '1.80', default),
        (DEFAULT, 'Theo Jose', '2006-05-31', 'M', '50.5', '1.50', default),
        (DEFAULT, 'Carlos Augusto Oswaldo', '1986-07-10', 'M', '115.5', '1.92', default);
