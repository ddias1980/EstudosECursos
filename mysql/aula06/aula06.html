Aula 06

Alterando a estrutura da tabela

Adicionando nova coluna


            ALTER TABLE pessoas ADD COLUMN profissao VARCHAR (10);

Removendo uma coluna, use DROP


            ALTER TABLE pessoas DROP COLUMN profissao;

Adicionando uma coluna na ordem certa

No exemplo, após a coluna nome, basta usar AFTER


            ALTER TABLE pessoas ADD COLUMN profissao VARCHAR(10) AFTER nome;

IMPORTANTE:Não existe BEFORE, então se quiser colocar uma coluna no começo da tabela, usa-se FIRST


            ALTER TABLE pessoas ADD COLUMN profissao VARCHAR(10) FIRST

Alterando a estrutura de uma coluna

Troque ADD por MODIFY

             	ALTER TABLE pessoas MODIFY COLUMN profissao VARCHAR(30);

Modficando o nome da coluna

E tipos primitivos, usa-se CHANGE


            ALTER TABLE pessoas CHANGE COLUMN profissao prof varchar(20);

Renomear a table inteira

Usa-se RENAME TO


            ALTER TABLE pessoas RENAME TO clientes;

Adicionando uma coluna de chave primária

Não dá pra adicionar na hora, primeiro, tem que adicionar a coluna e depois a chave primária


            ALTER TABLE cursos ADD COLUMN id int FIRST;
            ALTER TABLE cursos ADD PRIMARY KEY (id);
            ALTER TABLE cursos CHANGE COLUMN id id int not null auto_increment;

Apagando a tabela


            DROP TABLE cursos;
